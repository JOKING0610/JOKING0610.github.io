第一步：通过组策略赋予修改权限（最常用）

Win7 家庭版可能没有组策略功能，专业版/旗舰版/企业版可按此操作。

1. 点击「开始」菜单，在搜索框中输入 gpedit.msc，按回车。
2. 依次展开：计算机配置 → Windows 设置 → 安全设置 → 本地策略 → 用户权限分配。
3. 在右侧列表中找到 更改系统时间，双击打开。
4. 点击 添加用户或组，在输入框中输入 Administrators（或你当前使用的用户名），点击「确定」。
5. 一路点击「确定」保存，注销当前账户或重启电脑后生效。

第二步：检查 Windows Time 服务

如果组策略改完后还是无法修改，可能是时间服务被禁用了。

1. 按 Win + R，输入 services.msc，回车。
2. 在服务列表中找到 Windows Time，双击打开。
3. 将「启动类型」设为 自动，点击「启动」按钮，然后「确定」。

第三步：直接修改系统时间

完成上述设置后，点击任务栏右下角时间，选择「更改日期和时间设置」，将日期调回今天的准确日期（2026年3月26日）。

如果以上方法仍然无法修改，可以在「运行」中输入 cmd，以管理员身份打开命令提示符，尝试输入 date 2026-03-26 和 time 14:30:00 来强制修改。
